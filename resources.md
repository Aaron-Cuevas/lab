# Research resources

Curated, legitimate, mostly-free sources. No paywalled-bypass sites.

## Find papers (free / open access)
- arXiv — https://arxiv.org  (physics/optics preprints)
- Google Scholar — https://scholar.google.com
- Semantic Scholar — https://www.semanticscholar.org
- NASA ADS — https://ui.adsabs.harvard.edu  (astrophysics/physics literature)
- DOAJ — https://doaj.org  (Directory of Open Access Journals)
- PubMed Central — https://www.ncbi.nlm.nih.gov/pmc  (open access)
- CORE — https://core.ac.uk  (open access aggregator)
- Unpaywall — https://unpaywall.org  (legal free versions; browser extension)
- OpenAlex — https://openalex.org  (open scholarly index)

## Discover / connect ideas
- Connected Papers — https://www.connectedpapers.com  (visual citation graphs)
- Papers With Code — https://paperswithcode.com  (papers + their code)
- Litmaps — https://www.litmaps.com

## Math & equations reference
- NIST DLMF — https://dlmf.nist.gov  (Digital Library of Mathematical Functions)
- Wolfram MathWorld — https://mathworld.wolfram.com
- Wikipedia: List of math symbols — https://en.wikipedia.org/wiki/Glossary_of_mathematical_symbols

## LaTeX tools
- Detexify — https://detexify.kirelabs.org  (draw a symbol -> get the command)
- Overleaf docs — https://www.overleaf.com/learn  (LaTeX reference)
- TikZ examples — https://texample.net
- CTAN — https://ctan.org  (packages)

## Open courses / textbooks
- MIT OpenCourseWare — https://ocw.mit.edu
- OpenStax — https://openstax.org  (free textbooks: physics, calculus...)
- Feynman Lectures (online) — https://www.feynmanlectures.caltech.edu

## Data, code & reproducibility
- Zenodo — https://zenodo.org  (archive data/code, get a DOI)
- OSF — https://osf.io  (Open Science Framework)
- Zotero — https://www.zotero.org  (free reference manager)

## Optics / photonics specific
- Optica Open — https://opg.optica.org/open  (open-access optics)
- SPIE Digital Library — https://www.spiedigitallibrary.org  (some open)
